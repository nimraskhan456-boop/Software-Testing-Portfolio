# Software Testing Portfolio

QA Automation and Manual Testing Portfolio including test cases, bug reports, security testing, and automation scripts using Selenium and OWASP ZAP.

##  Security Testing Summary

Security testing was performed using OWASP ZAP with the following observations:

- Missing Security Headers (Medium)
- Informational security alerts generated
- XSS payload tested (`<script>alert(1)</script>`) — not executed
- Input validation verified manually
